MyAgenda - Gerenciador de Contatos Seguro
A MyAgenda é uma aplicação web desenvolvida com Django para o gerenciamento eficiente e seguro de contatos pessoais. O projeto foi desenhado para oferecer uma interface intuitiva onde usuários podem organizar seus círculos sociais e profissionais com total privacidade.

Objetivos do Projeto
Privacidade de Dados: Implementação de isolamento por usuário (cada usuário gerencia apenas seus próprios contatos).

CRUD Completo: Cadastro, visualização, edição e exclusão de registros.

Filtros e Busca: Localização rápida de contatos por nome, categoria ou telefone.

Ambiente Moderno: Uso de containers para garantir portabilidade e consistência entre ambientes.

Tecnologias Utilizadas
Backend: Django (Python)

Banco de Dados: Sqlite

Ambiente: Docker & Docker Compose

Segurança: Django Authentication System & CSRF Protection

Funcionalidades
[x] Autenticação de Usuários: Registro e login para proteção dos dados.

[x] Gestão de Contatos: Campos para Nome, Telefone, E-mail, Descrição e Categoria.

[x] Categorização: Separação de contatos por tags (Ex: Trabalho, Família, Amigos).

[x] Busca Dinâmica: Campo de pesquisa para filtrar a lista de contatos em tempo real.

[ ] Exportação: Funcionalidade para exportar contatos em CSV/PDF (Em breve).
